## 🛠️ Tumpukan Teknologi
- **Framework:** React Native (Expo SDK 50)
- **Navigasi:** Expo Router
- **Bahasa Pemrograman:** TypeScript

## 🚀 Cara Menjalankan
1. Kloning repositori ini.
2. Jalankan `npm install`.
3. Jalankan `npx expo start`.
